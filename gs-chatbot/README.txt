## Sobre o Projeto
O NeoSpherebot é um assistente virtual desenvolvido com IBM Watson 
Assistant e Node-RED para o tema "Sistemas de Previsão 
Climática e Prevenção de Desastres" da Global Solution 2026.

O bot responde perguntas sobre alertas climáticos, previsão 
do tempo, orientações de segurança, 
simulando um sistema real de monitoramento via satélite.

## Funcionalidades
- Consulta de alertas ativos por região
- Previsão do tempo
- Orientações de segurança por tipo de desastre

## Tecnologias
- IBM Watson Assistant (fluxo de diálogo)
- Node-RED (integração)
- Web Speech API (STT e TTS pelo telegram)
- Telegram Bot API
