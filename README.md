NEOSPHERE - DASHBOARD INTERATIVO DE PREVISÃO CLIMÁTICA E PREVENÇÃO DE DESASTRES

NeoSphere - Frontend Dashboard

1. Descrição Geral
O NeoSphere é um sistema inteligente de previsão climática focado na prevenção de desastres naturais. Este repositório contém o Front-end da aplicação, desenvolvido para monitorar as condições meteorológicas em tempo real na cidade de Tóquio, Japão. A interface conta com um dashboard interativo, um mapa de calor indicando áreas de risco (cores quentes para maior perigo e frias para menor perigo), gráficos dinâmicos de temperatura e probabilidade de desastres, além da interface visual de um chatbot para orientações de segurança.


2. Tecnologias Utilizadas
Para o desenvolvimento da interface, foram selecionadas tecnologias que garantem performance, responsividade e alta capacidade de visualização de dados geográficos:

HTML5: Utilizado para a estruturação semântica de toda a aplicação.

CSS3: Responsável pela estilização avançada, layout responsivo e implementação do Dark Mode.

JavaScript Vanilla: Empregado na lógica de manipulação do DOM e integração de componentes.

Leaflet.js: Biblioteca utilizada para a renderização do mapa interativo de Tóquio e camadas de calor.

Chart.js: Motor de renderização para a geração de gráficos dinâmicos de dados climáticos.

VS Codium: Ambiente de desenvolvimento principal utilizado para a escrita e depuração do código.


3. Estrutura de Pastas
A organização do projeto segue o padrão de arquitetura para aplicações client-side, facilitando a manutenção e escalabilidade do código:

/ (Raiz): Contém o arquivo index.html (ponto de entrada da aplicação) e o README.md (documentação técnica).

/css: Armazena o arquivo style.css, contendo todas as regras de design e variáveis de cores.

/js: Diretório do arquivo script.js, onde reside a lógica de inicialização do mapa, configuração dos gráficos e comportamento do chatbot.
